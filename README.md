# Network_setup_and_device_security_configuration
Цей репозиторій містить звіти та матеріали до лабораторних робіт з дисципліни, що викладається на кафедрі кібербезпеки та захисту інформації Київського національного університету імені Тараса Шевченка.

--Вміст репозиторію-- 

Лабораторна робота №1: 
1. Побудова простої мережі   
2. Проєктування базової топології мережі Ethernet. 
3. Налаштування статичних IP-адрес на кінцевих вузлах (ПК).  
4. Базова конфігурація комутаторів Cisco 2960 (призначення імені пристрою, встановлення локальних паролів, створення банера MOTD). 
5. Перевірка працездатності та зв'язку між вузлами за допомогою утиліти ping

Лабораторна робота №2:
1. Налаштування адреси для роботи з комутатором
2. Конфігурація віртуального інтерфейсу комутатора (SVI) для VLAN 1.
3. Налаштування віртуальних ліній VTY для забезпечення доступу.
4. Використання протоколу Telnet для віддаленого керування та адміністрування комутатора.
5. Діагностика мережевих підключень та тестування доступності віддалених вузлів.

Інструменти:
1. Cisco Packet Tracer — симуляція мережевої топології
2. Cisco IOS 15.0 / 15.2 — операційна система комутаторів і маршрутизаторів
3. Обладнання: Cisco 2960 (комутатор), Cisco 1941 (маршрутизатор)
