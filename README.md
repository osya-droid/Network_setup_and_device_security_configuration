# Network_setup_and_device_security_configuration

Навчальні лабораторні роботи з дисципліни «Комп'ютерні мережі»
КНУ імені Тараса Шевченка, Кафедра кібербезпеки та захисту інформації,  2026

Лабораторна 1 — Побудова простої мережі
Побудова базової Ethernet-топології (2 комутатори Cisco 2960, 2 ПК) у Cisco Packet Tracer. Основна увага — на фізичній та логічній організації мережі, статичній IP-адресації та базовому захисті доступу до пристроїв через CLI: паролі консолі, VTY і привілейованого режиму (enable secret).

Лабораторна 2 — Налаштування адреси для роботи з комутатором
Налаштування SVI-інтерфейсу (VLAN 1) для віддаленого управління комутатором по Telnet. Перевірка зв'язності між хостом і пристроєм, аналіз стану інтерфейсів командами show ip interface brief та show running-config.

Навички, що відпрацьовані:
1. Робота з Cisco IOS CLI: переміщення між режимами, читання конфігурації
2. Розмежування доступу до мережевого обладнання (console / VTY / enable secret)
3. Налаштування керованого комутатора для віддаленого адміністрування
4. Діагностика мережі: ping, ipconfig, show команди
5. Розуміння зв'язку між фізичним підключенням і станом протоколу

Інструменти:
1. Cisco Packet Tracer — симуляція мережевої топології
2. Cisco IOS 15.0 / 15.2 — операційна система комутаторів і маршрутизаторів
3. Обладнання: Cisco 2960 (комутатор), Cisco 1941 (маршрутизатор)
